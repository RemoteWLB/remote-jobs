**Overzicht  
** Je wordt lid van een team, bestaande uit experts in AI/data-annotatie, om trainingsdata te creëren voor 's werelds meest geavanceerde AI-modellen. Voorgaande ervaring met AI/data-annotatie is geen vereiste.. Dit is een unieke kans om een belangrijke speler te worden in de huidige AI/LLM-ruimte. We zoeken software engineers met 3+ jaar ervaring voor het trainen van grote AI-taalmodellen en het verbeteren van bestaande geavanceerde generatieve AI-modellen De projecten omvatten doorgaans uiteenlopende, variabele taken waarbij je deze AI-modellen helpt bij het leren coderen. Momenteel hebben we meer dan 200 openstaande vacatures!

 **Wat zijn jouw taken?**

  * Het evalueren van de kwaliteit van, door AI-gegenereerde code, inclusief het geven van menselijke, leesbare samenvattingen..

  * Programmering/Code-gerelateerde taken oplossen en functionele en efficiënte code schrijven.

  * Robuuste testcases ontwikkelen om te bevestigen dat de code effectief en efficiënt werkt.

De technische projectmanager geeft meer details: "Het gaat om het oplossen van code-gerelateerde problemen, het maken van instructies om anderen te helpen, en het beoordelen van code voordat het in het model wordt verwerkt.Er is veel variatie in de projecten, het varieert van ‘Welke Python-code is beter?’ tot ‘Maak een complete, mobiele applicatie met behulp van deze chatbot, en verbeter daarna de reacties van de chatbot om deze sneller te maken.’”
