### Job Summary

A Senior Software Engineer for .Net is an associate at Creative Chaos who is mainly responsible for developing server applications of various software platforms including but not limited to writing Restful APIs to be consumed by client applications. A senior software engineer makes sure that the functionality, security and performance of the server application is as per the design and is completely bug free. Senior engineers have a responsibility to mentor their juniors and build team strength through training.
