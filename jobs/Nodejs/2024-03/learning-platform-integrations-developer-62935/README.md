### Mission

The Global Learning Platform Integrations Developer will be responsible for developing and maintaining any code associated with integrations between Learning Platforms including but not limited to the learning management system (LMS), lab platforms, and productivity tools. The Learning Platform Integrations Developer will build relationships with stakeholders within LE to automate manual processes and ensure the most efficient and effective systems are available to create curriculum for our customers. This role will collaborate with cross functional stakeholders and external platform vendors, define business requirements, create and maintain the code base for any automations, and continuously seek to improve the user experience for any customers using said automations.

### Outcomes

  * Develop and maintain automation for the business to enable curriculum development and training delivery
  * Create API-based solutions to integrate solutions between learning platforms
  * Identify and solve manual processes to automate
  * Build relationships with the organization and adjacent business units to solve larger problems as a team
  *  **Must work US-hours (7 p.m.- 4 a.m.)**

### Competencies

  * 4-year degree (or equivalent experience) in Computer Science or related field with demonstrated ability
  * 3+ years of experience in public cloud technologies (AWS, Google Cloud, or Azure)
  * Proven experience in CICD tools (Source Control, Build, Deploy)
  * Ability to solve high complexity issues
  * Ability to communicate across all teams within the organization to provide solutions
  * Ability to take a given problem and complete a solution
  * Extensive development experience with JavaScript, CSS, Node.js, Python, and other API technologies. 
  * Know the installation, configuration, platform operations, and troubleshooting of API products
  * Experience in API design and creating RESTful API patterns
  * Understanding of learning management systems (LMS) would be a good to have
