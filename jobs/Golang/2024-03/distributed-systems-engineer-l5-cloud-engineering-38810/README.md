### Infra API

InfraAPI is Netflix's in-house IaaC solution - a resource federation platform built upon Kubernetes. We enable infra providers like compute, storage or networking to create end developer "resources" (aka primitives / abstractions) using a simple declarative language. This makes it possible to quickly ship experiences continuously raising the level of abstraction without needing to manually integrate with a number of delivery /mgmt platforms.
